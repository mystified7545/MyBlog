== Description ==
A flexible theme by ITstar with sticky widgets and jquery effects.

Requires WordPress 3.6.1 and PHP 5.

**Current Features**
* Ajax Login Form
* Focus on a special Comment
* Focus on a special Article
* Flexible (200px to 1600px Screen view)
* Manual Header Image
* Manual Background Image & Color
* 2 Widget Area on sidebar (Noraml , Sticky)
* 3 Widget Area on Footer
* A Navigation Widget
* Support Goolge Fonts
* 9 Theme color Style
* Lazy Content Option

== Installation ==

1. Upload 'fixy' to the '/wp-content/themes/' directory,
2. Activate the theme through the 'theme' menu in WordPress.
